How it works

1

Install

Clone the repo, run npm install && npm run build. Load the LISP bridge in AutoCAD.

2

Configure

Add autocad-mcp to your Claude Desktop config. It connects via MCP protocol.

3

Activate

Set your license key. It auto-binds to your machine on first launch — no setup needed.

4

Draw

Tell Claude what to draw. It handles the rest in AutoCAD.

Frequently Asked Questions

How does the license work?

Set your license key as an environment variable. On first launch, it automatically activates and binds to your machine. No extra steps needed.

Can I use one key on multiple machines?

No. Each key locks to the first machine it's activated on. If you need multiple machines, you'll need multiple keys.

What happens when my license expires?

The tools stop working until you renew. Your AutoCAD drawings are never affected.

Is the code open source?

The code is viewable for transparency, but a valid license key is required. Keys are validated through our activation server — they cannot be generated or bypassed locally.
